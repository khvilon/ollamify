# Profile

**Language:** **English** | [Русский](profile.ru.md)

Path in UI: `/profile`

## Purpose

- See your account info and activity
- Switch UI theme (light/system/dark)

## What you see

- **Account**
  - email / username
  - role (administrator vs user)
  - member since
  - last activity
- **Usage statistics**
  - projects count
  - API keys count
  - requests/month
  - storage used (MB)
  - total requests
- **Recent activity**
  - method + path + response time
  - relative time (“Just now”, “10 min ago”, ...)

## Theme

Theme is stored in `localStorage` (`theme` key) and applied after page reload.
