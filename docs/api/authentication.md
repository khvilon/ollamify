# Authentication

**Language:** **English** | [Русский](authentication.ru.md)

Ollamify uses a single gateway (`www3` / Nginx) and enforces auth for most `/api/*` endpoints.

You always send:

```
Authorization: Bearer <TOKEN>
```

Where `<TOKEN>` is either:
- a **JWT** (web UI login), or
- an **API key** (for external integrations)

## JWT flow (web UI)

1) Login:

- `POST /auth/login`
- body:

```json
{ "email": "admin@example.com", "password": "admin" }
```

- response:

```json
{ "token": "..." }
```

2) Use it:

```
Authorization: Bearer <token>
```

## API key flow (external integrations)

API keys are created in the web UI:

- Users → API keys → Create

Then use the API key exactly like a token:

```
Authorization: Bearer <api_key>
```

> External docs (generated by Zeus): `http://localhost/api/docs`
